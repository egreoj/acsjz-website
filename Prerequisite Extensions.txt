🔧 Must-Have Extensions for Web Development

Auto-refresh browser as you code.

🔎 Search: Live Server

✅ Purpose: Real-time preview without manual refresh

Auto-format HTML, CSS, JS neatly

🔎 Search: Prettier - Code formatter

✅ Purpose: Indentions, spacing, clean code layout

✅ 3. HTML CSS Support – ecmel
Autocomplete and IntelliSense for HTML and inline CSS

🔎 Search: HTML CSS Support

✅ Purpose: Smarter hints in HTML for CSS classes

✅ 4. IntelliSense for CSS class names in HTML – Zignd
Shows class names from CSS file while typing in HTML

🔎 Search: IntelliSense for CSS class names in HTML

✅ Purpose: No need to memorize class names

✅ 5. Tailwind CSS IntelliSense – Tailwind Labs
Kung gagamit ka Tailwind later

🔎 Search: Tailwind CSS IntelliSense

✅ Purpose: Auto-suggest utility classes with docs pop-up

✅ 6. Auto Rename Tag – Jun Han
Automatically renames closing tag when you change the opening tag (and vice versa)

🔎 Search: Auto Rename Tag

✅ Purpose: Save time on tag edits

✅ 7. Bracket Pair Colorizer 2 – CoenraadS
Makes matching brackets colorful

🔎 Search: Bracket Pair Colorizer 2

✅ Purpose: Easier to read nested HTML, JS, CSS

✅ 8. Path Intellisense – Christian Kohler
Autocomplete paths to files like images, CSS, JS

🔎 Search: Path Intellisense

✅ Purpose: Avoid path errors

✅ 9. Color Highlight – naumovs
Shows real colors for hex codes in CSS/HTML

🔎 Search: Color Highlight

✅ Purpose: Visual color preview as you type

✅ 10. Material Icon Theme – Philipp Kief
Nice folder/file icons

🔎 Search: Material Icon Theme

✅ Purpose: Better visual navigation

⚙️ Optional (for Advanced Users)
ESLint – for JavaScript/React projects

Firebase Tools – if using Firebase later

GitLens – for Git version control

REST Client – for testing APIs
